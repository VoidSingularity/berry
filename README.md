# Berry Loader

Berry Loader is a mod loader for Minecraft, designed for extreme extensibility.

## Build and Development

As of July 2026, Berry Loader has migrated to Gradle build system. Berry Loader uses the plugin
net.mcberry.stem for Minecraft mod development. Many thanks to Nik0chuu and LLM (I don't know which model)
for developing the plugin.

In order to build and run Berry Loader, you first have to run the `setup` task. After that, you should
be able to use `build` for building, and `runClient` and `runServer` for running Minecraft with Berry
Loader.
